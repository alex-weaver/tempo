Detailed Description

Represents a temporal scope which provides a lifetime for continuous expressions. Each TemporalScope object contains a Lifetime unique to the scope, and a TaskQueue for the thread the scope is to execute on. The Lifetime object allows continuous expressions to be notified when the scope has ended.
