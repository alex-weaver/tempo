static void Tempo.Threading.RunThreadWhilst ( ICellRead< bool >  threadActivity,
Action  constructor 
)
inlinestatic

A combination of the Whilst and Thread operations, provided as a single method for code clarity. This is equivalent to calling Thread() inside a Whilst() operation. Note that this is slightly different to reversing the order and calling Whilst() inside a Thread() operation. The former is used because it does not require any synchronization of the threadActivity argument, and minimizes resource usage when the threadActivity cell is false.

Parameters
threadActivityThe condition cell.
constructorThe constructor for the thread's top-level scope.
