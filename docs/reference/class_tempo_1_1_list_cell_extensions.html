static IListCellRead<TOut> Tempo.ListCellExtensions.WithEach< TIn, TOut > ( this IListCellRead< TIn >  source,
Func< TIn, ICellRead< TOut >>  selector 
)
inlinestatic

Creates a new temporal scope for each element in a list cell. If new elements are added to the list, new scopes are initialized. Likewise, when an element is removed from the list its scope ends. All element scopes end when the calling temporal scope ends. This behaviour is also combined with map-like behavior, where a function is applied to each element in the list, and the resulting list is this method's return value.
