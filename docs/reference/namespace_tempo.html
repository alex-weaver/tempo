class  ContinuousObject
 Provides a way for a continuous block to be constructed imperatively. ContinuousObject encapsulates a continuous block. Inheritors should call InitScope in the constructor. This will activate the inner block. The block is ended when the ContinuousObject's reference count reaches zero. Ineritors should implement the WhileAlive method, which is active when the inner scope is active. Inheritors may override Destroy to perform cleanup when the object is destroyed.
