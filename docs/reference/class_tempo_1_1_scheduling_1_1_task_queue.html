IUpdateableHandle Tempo.Scheduling.TaskQueue.CreateUpdateable ( Lifetime  lifetime,
Action  handler 
)
inline

Constructs a new updateable with the given handler. Calling NeedsUpdate() on the return value will schedule the handler to be run. Until the handler is actually executed, any further calls to NeedsUpdate will have no effect.

Parameters
lifetimeThe lifetime of the updateable. When the lifetime has ended, the handler will not be scheduled and further calls to NeedsUpdate will have no effect.
handlerThe action to perform on each update.
Returns
A handle to allow the caller to schedule updates.

Schedule a transactional action. This method will attempt to lock all of the participants. If they can be locked, the action will be scheduled immediately. Otherwise, it will be put in a queue until all participants are available.
