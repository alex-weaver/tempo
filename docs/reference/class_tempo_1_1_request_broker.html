Detailed Description

RequestBroker keeps a list of all concurrent requests made via continuous calls to ApplyRequest or ApplyConstRequest, and each time the list changes, a single aggregate of the list is computed. This is useful for sharing access to an exclusive resource.

Template Parameters
TRequestThe type of the request objects.
TAggregateThe type of the resulting aggregate object.
