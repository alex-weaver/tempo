Detailed Description

Describes a single change of a list cell. If Action is Add, then NewItems contains the items added and NewStartingIndex is a valid index. If Action is Removed, then OldStartingIndex and OldItemCount define the range that was removed If Action is Replace, then NewItems contains the replacement elements. OldStartingIndex and NewStartingIndex are equal, and are valid indices. OldItemCount is equal to NewItems.Count()

ListCellChanged implements IRefCounted in case the NewItems collection contains reference counted items. When the ListCellChanged object is destroyed, the items in the NewItems collection are released.
