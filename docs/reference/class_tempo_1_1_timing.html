inlinestatic

Schedule a sequential block to execute after a specific interval has elapsed. Note that the interval is a lower bound on when the handler will be exectued, not an exact guarantee. If the scope ends before the interval has elapsed, the handler will not be executed.

Parameters
intervalThe interval to wait before scheduling the handler.
handlerThe sequential block to execute.
